# Gerador de Senhas Seguras

Este é um projeto desenvolvido em Python com o objetivo de gerar senhas seguras de forma simples e prática. O projeto foi criado como parte de um estudo de desenvolvimento, versionamento com GitHub, testes automatizados e boas práticas de documentação.

## 🎯 Objetivo

Gerar senhas aleatórias e seguras, garantindo que cada senha contenha:
- 1 letra maiúscula
- 1 letra minúscula
- 1 número
- 1 símbolo

## 🛠️ Tecnologias Utilizadas

- Python 3.12+
- Pytest (testes automatizados)
- Git e GitHub

## ⚙️ Funcionalidades

- Solicita ao utilizador o tamanho desejado para a senha (mínimo de 8 caracteres).
- Validação de entrada para garantir apenas números inteiros.
- Geração da senha utilizando regras seguras de caracteres.

## 🚀 Como Executar o Projeto

1. **Clone o repositório** (se ainda não o tiver feito):
   ```bash
   git clone [https://github.com/LevisonFranciscattoFit/gerador-senhas.git](https://github.com/LevisonFranciscattoFit/gerador-senhas.git)
   cd gerador-senhas
Instale as dependências:

Bash

Bash

🧪 Como Executar os Testes
Para executar os testes automatizados que validam o tamanho e a complexidade da senha, execute no terminal:

Bash
